### 1.2 页面优先级

| 优先级 | 页面 | 说明 |
|--------|------|------|
| P0 | 练习会话页 | 核心交互，必须稳定 |
| P0 | 会话报告页 | 练习结束必看 |
| P1 | 首页 | 用户入口 |
| P1 | Agent 详情页 | 选择角色开始练习 |
| P1 | 历史记录 | 查看过往练习 |
| P2 | 管理后台 | 配置 Agent/角色/知识库 |

| P3 | 个人中心 | 用户设置 |

## 4. 弹窗/对话框规格汇总

| 弹窗名称 | 触发场景 | 内容 | 按钮 |

## 10. 设计规范参考

本文档中的页面设计遵循 `frontend-design-system.md` 中定义的 Modern Soft UI 设计系统：

- 背景: `bg-slate-50` (大背景), `bg-white` (卡片)
- 文字: `text-slate-900` (主标题), `text-slate-500` (次级)
- 圆角: `rounded-2xl` (卡片), `rounded-full` (按钮)
- 阴影: 弥散阴影 `shadow-[0_8px_30px_rgb(0,0,0,0.04)]`
- 毛玻璃: `bg-white/70 backdrop-blur-xl`

详细设计规范请参考 `.kiro/steering/frontend-design-system.md`。
