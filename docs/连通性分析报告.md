### 9.2 发现的问题（功能性Bug，非连通性问题）

⚠️ **以下问题解释了你遇到的"效果不好"：**

#### 问题1: AI联网搜索而非使用知识库

**根因**: Agent未正确绑定知识库时，AI会fallback到联网搜索

**代码位置**:
- `backend/src/sales_bot/services/voice_runtime_policy.py:43` - 默认`enable_web_search: False`
- `backend/src/sales_bot/websocket/stepfun_realtime_handler.py:1152` - 有提示词"禁止联网搜索"
- 但如果知识库未绑定，AI无法检索内部知识

**解决步骤**:
1. 在管理后台 → 知识库管理 → 创建知识库并上传文档
2. 在管理后台 → Agent管理 → 编辑Agent → 关联知识库
3. 确保Agent的`retrieval_priority`设置为`kb_only`或`kb_first`

#### 问题2: PPT内容不显示

**根因**: 可能是以下原因之一
- python-pptx库未安装
- PPT文件解析失败（损坏的PPT或不兼容格式）
- WebSocket事件未正确发送

**排查步骤**:
1. 检查后端日志是否有`[PARSE_ERROR]`或`[PPTX_NOT_AVAILABLE]`
2. 确认PPT文件是.pptx格式（.ppt旧格式可能不支持）
3. 检查前端WebSocket消息是否有`slide_update`事件

**代码位置**:
- `backend/src/presentation_coach/services/ppt_parser.py` - PPT解析服务
- `backend/src/presentation_coach/websocket/presentation_handler.py:1154` - 发送slide_update
- `web/src/components/practice/presentation/SlideViewer.tsx` - 前端渲染

#### 问题3: 微信登录按钮

- 问题: 无 onClick 处理函数
- 影响: 仅为展示性按钮
- 建议: 如需微信登录功能，后续实现
